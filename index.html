Booki

Ce premier projet me projetais dans le premier travail d'un développeur la transformation d'une maquette. Testant de cette façon ma compréhension du pouvoir structurant du Html et les possibilités de mise en page du Css, grâce à une page d'acceuil d'un site de reservation d'hôtel.

Ohmyfood

Ce projet avait pour objectif de tester ma capacité à rendre dynamique un site grâce aux animations Css. L'outil Sass m'a été présenté et ensuite je devais réaliser pour un site de réservation de plat, la transformation d'une maquette avec plusieurs animations.

La Panthère

Ici un travail assez différent, à partir d'un site existant je devais améliorer le référencement de ce site mais aussi son accessibilité. Je devais ensuite rédiger un rapport sur les axes d'améliorations choisis en motivant mes choix.

Kanap

Premier pas dans le développement dynamique d'un site, dans ce projet le Javascript débuta par la partie front-end. Dasn un site marchand nous devions réaliser la partie qui intéragie avec l'utilisateur jusqu'au passage de commande finale.

Piquante

Dans ce projet il nous est demandé de réaliser l'API d'un site communautaire de fan de sauce piquante. J'y ai appris à me servir d'une base de donnée, réaliser les chemins pour les différentes requêtes, nottament grâce à node.js.

Kasa

Enfin le dernier projet portait sur le site web-monopage. Grâce à React on m'a demandé de réaliser un site de location d'appartement entre particulier. En exploitant une base de données j'ai donc créé un site entièrement dynamique.
