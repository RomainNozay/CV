Projet Kanap

Javascript Front-end

Dans ce projet j'ai dû réaliser la partie dynamique d'un site de e-commerce de canapé". Afficher les produits dynamiquement sur la page d'accueil à partir d'une base de donnée, créer des pages personnalisées pour chaque produits. Utiliser le local storage pour transférer les données sélectionnées par l'utilisateur vers une page panier. Sur cette dernière faire apparaître un panier qui récupère les données du local storage et de la base de donnée. Pour finalement envoyer vers une base de donnée les choix de l'utisateur et vers l'utilisateur un numéro de commande. Vous pouvez trouver le projet sur Github à cette adresse
